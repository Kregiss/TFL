# Лабораторная работа №3

Мой язык:

$S \to SbTa$

$S \to ab$

$S \to ba$

$T \to SS$

$T \to bT$

-----------------------

## Анализ детерминированности

Во-первых, слово может начинаться либо с ab, либо с ba. Во-вторых, можно записать вид всех слов языка через нетерминал S так: $S \to (ab|ba)(bb^\*SSa)^\*$.

И из такой записи можно увидеть недетерминизм: 

$\[(ab)b^n\](bab)(ba)a$, - после прочтения $[(ab)b^n\]b$ должны начать снимать со стека для следующих ab

$\[(ab)b^n\](bbab)(ba)a$. - после прочтения $[(ab)b^n\]b$ должны сделать ещё один заход b без изменения стека
