# Лабораторная работа №3

Мой язык:

$S \to SbTa$

$S \to ab$

$S \to ba$

$T \to SS$

$T \to bT$

-----------------------

## Анализ детерминированности

Во-первых, слово может начинаться либо с ab, либо с ba. Во-вторых, можно записать вид всех слов языка через нетерминал S так: $S \to (ab|ba)(bb^\*SSa)^\*$.

И можно увидеть недетерминизм: 

$\[(ab)b^n\](ba)(ba)a$, - после прочтения $[(ab)b^n\]b$ должны начать снимать со стека для обработки ba

$\[(ab)b^n\](bbab)(ba)a$. - после прочтения $[(ab)b^n\]b$ должны сделать ещё заход в $b^\*$ без изменения стека

1. Обязательно проходим начало строки ab или ba
2. Если строка продолжается, то при проходе по обязательной b кладём на стек AAA - одна A для пометки, что надо закончить текущий S, а два остальных - надо закончить два вложенных SS.
3. При встрече любого a со стека снимается A, т.к. это означает, что либо встретилась подстрока ab, либо ba, и либо встретилась a, которая идёт после SS (закончили текущий S)

## Беспрефиксность
Очевидно, что язык небеспрефиксный, т.к. любое слово начинается с (ab|ba), при этом слова ab и ba лежат в языке.
